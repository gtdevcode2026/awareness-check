# newspaper / ensemble-logs

AI prompt + response logs for every build of the **newspaper** template (status: beta).

Each build creates a fresh timestamped subfolder. Files inside show the prompt that was sent and the response that came back.

Canonical copies of these logs also live in the project-root `ensemble-logs/` folder (untouched legacy path). This folder is the per-template mirror.

Folder kept under version control via this README (zip strips empty folders).
